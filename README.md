# Rack::SSI

Rack middleware for processing SSI based on nginx HttpSsiModule.
Directives currently supported: 'block' and 'include'

## Installation

Add this line to your application's Gemfile:

    gem 'rack_ssi'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rack_ssi

## Usage

    require 'rack_ssi'
